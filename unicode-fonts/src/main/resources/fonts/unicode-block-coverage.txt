This project uses the unicode-3.2 test page to test whether it supports most unicode characters. Since no font
supports all the characters, this project needs 16 fonts (~20MB in total, or ~10MB when zipped) to cover
almost all characters.

The following list shows which unicode block is covered by which font. I did a reasonable attempt to minimize the
total size, but there are a lot of characters...
